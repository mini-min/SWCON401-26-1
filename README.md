# 공간음향 피드백 기반 전방머리자세 교정 시스템
> A Forward Head Posture Correction System Based On Spatial Audio Feedback

- 2026-1 Sofware Convergence Capstone Design | 소프트웨어융합캡스톤디자인 (SWCON401)
- Period : 2026.03.05 - 2026.06.19
- Development Environment : SwiftUI, macOS 13.0+, CoreMotion, AVFoundation

### Overview
공간음향 피드백 기반 전방머리자세 교정 시스템은 AirPods의 Head Tracking 데이터를 활용하여 사용자의 머리 자세를 실시간으로 감지하고, 자세가 흐트러졌을 때 공간음향 피드백을 통해 자연스럽게 바른 자세로 돌아오도록 유도하는 macOS 애플리케이션입니다.

기존 자세 교정 서비스가 팝업, 알림, 경고음처럼 사용자의 작업 흐름을 직접적으로 방해하는 방식에 의존했다면, 본 프로젝트는 사용자가 듣고 있는 소리의 방향감과 거리감을 변화시켜 자세 이상을 감각적으로 인지하도록 설계했습니다.

## Background
장시간 노트북을 사용하는 환경에서는 사용자가 자연스럽게 고개를 앞으로 내밀거나 숙인 자세를 취하기 쉽습니다. 이러한 자세가 반복되면 전방머리자세, 흔히 말하는 거북목 증후군으로 이어질 수 있으며 목과 어깨의 피로, 경추 하중 증가, 만성 통증 등의 문제를 유발할 수 있습니다.

기존 자세 교정 시스템은 사용자의 자세를 감지한 뒤 알림이나 경고를 제공하는 방식이 많습니다. 하지만 반복적인 알림은 작업 몰입을 방해하거나 쉽게 무시될 수 있고, 자세를 감지하는 것만으로는 실제 교정 행동을 지속적으로 이끌어내기 어렵습니다.

본 프로젝트는 이 문제를 탐지 중심이 아닌 행동 유도 UX 중심으로 접근했습니다. 사용자가 별도의 알림을 확인하지 않아도 소리의 변화만으로 자신의 자세 이상을 인지하고, 자연스럽게 정자세로 돌아오도록 만드는 것이 핵심 목표입니다.

<br>

## System
본 시스템은 크게 세 단계로 동작합니다.

1. Head Tracking 데이터 수집
    AirPods의 IMU 센서를 통해 사용자의 머리 움직임을 실시간으로 수집합니다.
2. 자세 불량도 계산
    Pitch, Yaw, Roll 값을 바탕으로 사용자의 자세가 정자세 기준점에서 얼마나 벗어났는지 계산합니다.
3. 공간음향 및 시각 피드백 제공
    자세가 흐트러질 경우 음원의 방향감과 거리감을 변화시키고, 화면 오버레이를 통해 보조적인 시각 피드백을 제공합니다.
   
### 1. 자세 인식 알고리즘

본 시스템은 AirPods의 Head Tracking 데이터를 활용하여 사용자의 머리 움직임을 실시간으로 분석합니다.

| Axis | Description |
| -- | -- |
| Pitch | 고개의 앞뒤 숙임 정도 (X축이 해당) |
| Yaw | 고개의 좌우 회전 정도 (Y축이 해당) |
| Roll | 고개의 측면 기울임 정도 (Z축이 해당) |

시스템 실행 시 사용자가 처음 취한 자세를 개인별 정자세 기준점으로 설정합니다. 이후 실시간으로 수집되는 Head Tracking 데이터를 기준점과 비교하여 자세 불량도를 계산하고, 기준 범위를 벗어난 경우 자세 이탈 상태로 판단합니다.

### 2. 공간음향 피드백
정자세 상태에서는 음원이 사용자의 머리 중심에 위치하여 소리가 선명하고 균형 있게 들리도록 구성했습니다.

반면 사용자의 자세가 흐트러질 경우, 머리의 이탈 방향에 따라 음원의 위치와 거리감이 변화합니다. 예를 들어 고개가 앞으로 숙여지거나 한쪽으로 기울어지면 소리가 중심에서 벗어나거나 멀어지는 것처럼 들리도록 하여, 사용자가 자신의 자세 이상을 감각적으로 인지할 수 있도록 설계했습니다.

### 3. 시각적 피드백
청각 피드백만으로 부족할 수 있는 방향 정보를 보완하기 위해 화면 오버레이 기반의 시각 피드백도 함께 제공합니다. 사용자의 자세 이탈 방향과 정도를 화면상 위치 및 색상 변화로 표현하여, 현재 자세 상태를 더 직관적으로 파악할 수 있도록 했습니다.

## 결론
✔️ 기존 자세 교정 방식이 사용자의 자세 이상을 직접적으로 알려주는 데 집중했다면, 본 시스템은 사용자가 듣고 있는 소리의 방향감과 거리감을 변화시켜 스스로 자세 이상을 인지하고 교정하도록 설계했습니다.

✔️ 이를 통해 자세 교정 문제를 단순한 탐지 기술이 아닌, 행동 변화를 유도하는 UX 문제로 확장했다는 점에서 의의가 있습니다.

✔️ 향후에는 더 많은 사용자를 대상으로 장기적인 실험을 진행하여 실제 자세 개선 효과를 정량적으로 분석하고, 사용자별 신체 조건과 작업 환경에 따른 개인화 피드백 기능을 고도화할 계획입니다.
